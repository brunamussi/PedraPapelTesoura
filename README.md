# Pedra, Papel e Tesoura &#x1F48E; 	&#x1F9FB; &#x2702;&#xFE0F;

Este é um aplicativo que simula o jogo jokenpô contra um Bot. 

O APP possui duas telas tela com os seguintes conceitos:
* Layout
* Activity
* Arrays
* Estruturas Condicionais
* Switch
* Construtores
